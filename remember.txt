15.#include <stdio.h>

char* fun1(){
    char *str = "abcde";
    return str;
}

char* fun2(){
  static char s[] = "12345";
  return s;
}
int main(){
  char*ps;
  ps = fun1();
  printf("%s\n",ps);
  ps = fun2();
  printf("%s\n",ps);
  
}
在删除static后输出不了12345
因为数组的生命周期结束所占用的内存就会被释放，指针就会变为悬挂指针
但是第一个函数中指针指向的是静态变脸，在代码区，是全局存在的，所以不会被销毁，指针可用

16.注意函数指针的写法，int(*p)(int,int)而不是int*p(int,int)这就变成返回int指针类型的函数了；
