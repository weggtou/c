冒泡排序法
假如有5个数想排大小，19，3，46，23，5
可以每两个进行比较，将大数排在后面
19和3比，大数到后所以19和3交换位置，
3 19 46 23 5
然后19和46 比，不交换
46和23比 交换
46和5比交换
这样大数会沉到最底部
再将剩下的4个数排一次
直到最后连个数排
所以总共要进行n-1轮比较
第i次比较要进行n-i次比较

代码
int main(){
  int all = {19，3，46，23，5};

      if(a[i]>a[i+1]){
        int t;
        t = a[i];
        a[i]=a[i+1];
        a[i+1]=t;
      }
    }
  }
}
